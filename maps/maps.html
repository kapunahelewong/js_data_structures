Maps in JavaScript

Use JavaScript Console to see the map in action. Info about maps:

  • The keys are unique.
  • sometimes called an associative array or dictionary
  • fast look up of data based on key values
  • Unlike keys for an object (that can be strings or symbols), but keys for a Map can be any value (functions, numbers, objects, boolean, NaN, etc.).

Objects have traditionally served this purpose in JavaScript.
